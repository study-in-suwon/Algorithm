# 구현 알고리즘
- 머리속에 있는 알고리즘을 소스코드로 바꾸는 과정
- 완전탐색과 시뮬레이션 유형이 구현으로 묶어서 다뤄짐
  * 완전 탐색 : 모든 경우의 수를 모두 계산하는 방법
  * 시뮬레이션 : 문제에서 제시한 알고리즘을 한 단계식 차례대로 직접 수행하는 방법  
- 구현 알고리즘은 메모리 제약에 주의해야 한다.
